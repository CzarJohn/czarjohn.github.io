  • Tips

  • Getting the server's IP address

  • 1. Go to server (Sir Joseph's PC)

  • 2. Open a command prompt

  • 3. Enter 'ipconfig' (without the quotes)

  • 4. Get the ipv4 address

  • The app url would be ipaddress/cci


  • Starting the server manually

  • 1. Go to server (Sir Joseph's PC)

  • 2. Open wampserver from the start menu


  • Setting the server to start automatically

  • 1. Go to server (Sir Joseph's PC)

  • 2. Press windowskey + r

  • 3. Find wampapache64, right click, and go to properties

  • 4. Select automatic at startup type then press OK

  • 5. Repeat steps 3 and 4 for wampmysqlld64
